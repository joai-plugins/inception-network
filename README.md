# Inception Network

Connect Inception Network to Claude, Codex, and ChatGPT through JoAi's hosted MCP app server.

[Open app page](https://joai.ai/en/apps/inception-network) · [Browse all apps](https://joai.ai/apps)

## Works With

- ChatGPT
- Claude
- Codex
- Cursor
- Other MCP clients

## Install

### Claude

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-inception-network@joai-claude-plugins
```

### Codex

```bash
codex mcp add joai-inception-network --url https://cortex.joai.ai/mcp/apps/inception-network
```

### Cursor

Copy to local plugins:

```bash
git clone https://github.com/joai-plugins/inception-network.git ~/.cursor/plugins/local/joai-inception-network
```

Or use the [MCP deeplink](cursor://anysphere.cursor-deeplink/mcp/install?name=joai-inception-network&config=eyJtY3BTZXJ2ZXJzIjp7ImpvYWktaW5jZXB0aW9uLW5ldHdvcmsiOnsidXJsIjoiaHR0cHM6Ly9jb3J0ZXguam9haS5haS9tY3AvYXBwcy9pbmNlcHRpb24tbmV0d29yayJ9fX0=) to install directly.

### ChatGPT

Paste the MCP URL in ChatGPT settings:

```
https://cortex.joai.ai/mcp/apps/inception-network
```
